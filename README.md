## Contributting

### Requirements

- [Git](https://github.com/git-guides/install-git)
- [Npm](https://www.npmjs.com/get-npm)
- [Vitepress](http://vitepress.dev)

### Clone

Clone this repository and enter the project's directory.

```shell
git clone https://github.com/Power-Components/powergrid-doc.git && cd powergrid-doc
```

### Install

Run the installation script.

```shell
./run.sh
```
